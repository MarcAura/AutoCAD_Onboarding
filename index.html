Overview

This guide explains how to set up AutoCAD LT (2019 and 2026), how to install required fonts and plot styles, and how to use eTransmit to share drawings with all dependencies (fonts, CTBs, Xrefs, images, etc.) so files look identical on all systems. It also covers layer management, paper space layouts, a smart title block with attributes, the Block Editor, Sheet Set Manager, and how to produce multi‚Äësheet PDFs quickly.

Quick Reference (Most‚ÄëUsed Actions)

Update REV on All Sheets

  2. Basic: ATTEDIT REV on each sheet ‚Üí ATTSYNC.

  3. Regenerate: REGENALL (or REGEN for current viewport).

Freeze Non-Isolated Layers with LAYISO

LAYISO can do more than lock layers‚Äîby changing its settings you can freeze or turn off all non‚Äëselected layers.

  1. Type LAYISO in the command line and type S for settings.
    2.  [Off/Lock and fade/Lock and dim] <Lock and fade>:

Additional Layer Commands

Isolate Selected Layers

  1. Select objects you want to keep visible.
  2. Run LAYISO ‚Üí all other layers are temporarily hidden (frozen in view).

Freeze Layer by Selection

  1. Run LAYFRZ.
  2. Select any object ‚Üí its entire layer is frozen.

Thaw Frozen Layers

LAYTHAW restores (unfreezes) layers that were frozen with LAYFRZ or LAYISO.

Turn All Layers On

LAYON turns on all layers that were previously turned off.

Turn Off Layers

LAYOFF hides selected layers (they remain in the drawing but invisible).

Unlock Layers

LAYULK unlocks layers that were locked with LAYLCK.

Lock Layers

LAYLCK locks selected layers to prevent editing but still shows them in the drawing.

Merge Layers

LAYMRG merges selected layers into another target layer. The originals are deleted.

Delete Unused Layers

LAYDEL deletes selected layers (must be empty of objects first, except 0 and Defpoints).

Restore Previous Layer State

LAYUNISO restores all layers to the state before the last LAYISO.

Publish Engineering PDF + Film Layouts

  1. Engineering PDF:
    1. Run PUBLISH.
    2. Select all engineering layouts (exclude film layouts).
    3. Set Plotter = DWG To PDF.pc3.
    4. Set Sheet Size = 8.5x11 (or project standard).
    5. Check Single PDF output ‚Üí Save as Engineering Set.
  2. Film Layouts:
    1. Run PUBLISH again.
    2. Select only film layouts.
    3. Set Plotter = Film printer (or export device).
    4. Confirm Plot Style (CTB) = Film-specific (e.g., Film_Trace.ctb).
    5. Output each layout as needed for film production.

Create Layer State from Selection

  1. Turn everything on: LAYON
  2. Thaw all layers: LAYTHW
  3. Isolate layers of selected objects: LAYISO‚Üí [Settings] (at bottom of screen) ‚Üí Off ‚Üí Off ‚Üí select objects ‚Üí Enter
  4. Save this view as a preset: LAYERSTATE ‚Üí New (name it clearly, e.g., SILVER_TRACES_LAYOUT)
  5. Restore anywhere (Model or specific viewport): LAYERSTATE ‚Üí Restore
  6. Return to full drawing: LAYUNISO ‚Üí (and LAYTHW if needed)

Layouts (Paper Space)

Layouts represent printable sheets. Model Space = 1:1 design; Layouts = paper with border/title block + scaled viewports.

  1. Switch to a Layout tab.
  2. Insert or Xref your Title Block template.
  3. Create viewport (MV) and set scale (1:2, 1:5, etc.).
  4. Activate viewport, Freeze/Thaw per‚Äësheet layers, then restore the correct Layer State.

Each Layout becomes a page in the final PDF. Use different presets per Layout.

Viewports in AutoCAD

A viewport is a window in a Layout (Paper Space) that looks into Model Space. Use them to control what you see, how big it looks, and which layers are visible.

How to Use Viewports

  1. Switch to a Layout tab (Paper Space).
  2. Type MV (Make Viewport) or use the ribbon to create one.
  3. Draw a rectangle where you want the viewport.
  4. Double-click inside the viewport ‚Üí you‚Äôre now ‚Äúin‚Äù Model Space through that window.
  5. Set the desired scale (e.g. 1:1, 1:2) from the status bar.
  6. Use LAYFRZ or VPFreeze to hide layers in only that viewport.
  7. Double-click outside the viewport to return to Paper Space.

When to Use Viewports

  • To show the same drawing at different scales.
  • To isolate different layers in separate sheets.
  • To make multiple detail callouts from one master model.

‚úÖ Rule of thumb: Always draw in Model Space, always plot from Layouts, and use Viewports to control views.

Block Editor (BEDIT)

Edit reusable blocks (title block, symbols) in isolation and update all instances at once.

  1. Run BEDIT ‚Üí choose block (e.g., TITLE_BLOCK).
  2. Add/modify geometry, insert attributes, or place fields.
  3. Save & Close ‚Üí ATTSYNC to refresh placed blocks.

Attributes

Attributes are editable text fields inside a block (e.g., REV, PROJECT, DRAWN_BY, DATE). They can be edited and updated across Layouts.

Create Attributes
  1. Command: ATTDEF
  2. Fill in:
    • Tag (e.g., REV, PROJECT, DRAWN_BY)
    • Prompt (shown when editing)
    • Default (e.g., A)
  3. Place the attribute inside the title block.
Edit & Sync
  • ATTEDIT / EATTEDIT ‚Äî edit values
  • BATTMAN ‚Äî batch manage attributes
  • ATTSYNC ‚Äî sync block definition & instances

Title Block with Attributes

  1. Draw border/boxes/logo in Paper Space.
  2. Create attributes with ATTDEF: REV, PROJECT, DATE, DRAWN_BY.
  3. Select all ‚Üí BLOCK ‚Üí name TITLE_BLOCK.
  4. Insert into each Layout and fill fields.

Appendix A ‚Äî Updating Revision Numbers (A ‚Üí C)

  1. ATTEDIT (or EATTEDIT).
  2. Select the title block attribute REV.
  3. Change value from A ‚Üí C.
  4. Repeat in each Layout if not linked.

  2. Right‚Äëclick Sheet Set name ‚Üí Properties.
  3. Edit CurrentSheetRevision from A ‚Üí C.
  4. OK ‚Üí REGENALL.

  1. BEDIT title block ‚Üí select REV attribute.
  2. Insert Field ‚Üí SheetSet ‚Üí CurrentSheetRevision.
  3. Save & Close ‚Üí ATTSYNC.

Appendix C ‚Äî Worked Example (Existing + New)

Part 1: Organizing an Existing Design

Scenario: Drawing has traces, fiducials, adhesive outlines, and notes mixed on random layers.

Goal: Reorganize geometry, apply Layer States, build Layouts, insert Title Block with attributes, publish multi‚Äëpage PDF.

  2. Create Standard Layers ‚Äî LAYER:

  3. Move Objects ‚Äî use MATCHPROP or manual reassignment.
  4. Create Layer States ‚Äî LAYERSTATE:

    • FINAL_ASSEMBLY ‚Üí ALL layers
  5. Create Layouts ‚Äî insert Title Block; viewport; scale; restore appropriate Layer State. Rename:
    • Layout ‚ÄúSilver Traces‚Äù
    • Layout ‚ÄúAdhesive‚Äù
    • Layout ‚ÄúFinal Assembly‚Äù
  6. Insert Title Block Attributes ‚Äî REV/PROJECT/DRAWN_BY/DATE ‚Üí ATTSYNC; set REV = A.

  1. Start from Master Template ‚Äî ACAD-2019-LITE-MASTER TEMPLATE_10-16-20.

  3. Create Layer States ‚Äî mirror Part 1 for consistency.
  4. Create Layouts ‚Äî Title Block + viewport + restore state for each sheet.
  5. Fill Title Block Attributes ‚Äî e.g., PROJECT = ‚ÄúFHE Smart Label‚Äù, REV = A, DATE = today, DRAWN_BY = engineer name.
  6. Optional (Advanced): Link to Sheet Set ‚Äî SHEETSET; add Layouts; link REV to CurrentSheetRevision.

Part 3: Benefits of This Workflow
  • Consistency: same structure across drawings.
