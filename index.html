Overview

This guide explains how to set up AutoCAD LT (2019 and 2026), how to install required fonts and plot styles, and how to use eTransmit to share drawings with all dependencies (fonts, CTBs, Xrefs, images, etc.) so files look identical on all systems. It also covers layer management, paper space layouts, a smart title block with attributes, the Block Editor, Sheet Set Manager, and how to produce multi‑sheet PDFs quickly.

Quick Start (Most‑Used Actions)

Update REV on All Sheets

  1. Preferred: SHEETSET → Sheet Set Properties → Change CurrentSheetRevision → REGENALL.
  2. Basic: ATTEDIT REV on each sheet → ATTSYNC.

Switch Layer Set for a Layout

  1. Activate viewport (double‑click inside).
  2. Run LAYERSTATE → choose saved state (e.g., SILVER_TRACES_LAYOUT).

Publish All Layouts → PDF

  1. Run PUBLISH → select layouts → output to single PDF.

Tip: Toggle on‑screen lineweights with LWDISPLAY → ON. Toggle OSNAP with F3; grid snap with F9.

Onboarding Essentials

Install fonts (TTF/OTF), copy SHX/CTB, set default template, then restart AutoCAD.

AutoCAD Onboarding Instructions
  • Retrieve files from a co‑worker or from the Onboarding folder. Note: WIN+R → %AppData% to quickly find hidden folders.
  • Retrieve all Plot Styles (CTB). Copy into:
    C:\Users<UserName>\AppData\Roaming\Autodesk\AutoCAD LT 20XX\enu\Plotters\Plot Styles
  • Install all Windows fonts (.ttf or .otf) by right‑clicking each and selecting Install. Required examples:
    • TT0142M.ttf
    • TT0200M.ttf
    • TT0201M.ttf
    • TT0202M.ttf
  • Copy all AutoCAD SHX fonts into:
    C:\Program Files\Autodesk\AutoCAD LT 20XX\Fonts
  • Copy templates into:
    C:\Users<UserName>\AppData\Local\Autodesk\AutoCAD LT 20XX\R32\enu\Template
  • Restart computer. Open AutoCAD and select the template: ACAD-2019-LITE-MASTER TEMPLATE_10-16-20.

Paths (copy)

  • Plot Styles (CTB) 2019 → C:\Users\<User>\AppData\Roaming\Autodesk\AutoCAD LT 2019\enu\Plotters\Plot Styles
  • Plot Styles (CTB) 2026 → C:\Users\<User>\AppData\Roaming\Autodesk\AutoCAD LT 2026\R32\enu\Plotters\Plot Styles
  • AutoCAD SHX → C:\Program Files\Autodesk\AutoCAD LT 20XX\Fonts
  • Windows Fonts → C:\Windows\Fonts
  • Templates (2019/2026) → C:\Users\<User>\AppData\Local\Autodesk\AutoCAD LT 20XX\R32\enu\Template

eTransmit (Full Package)

Transmitter Short Checklist
  1. Open DWG in LT 2019LT 2026.
  2. Type ETRANSMIT → Enter.
  3. Select Full_Transmit_With_Fonts setup (NOT Standard).
  4. If creating a new setup, check every include option:
    • Include fonts
    • Include plot style tables
    • Include all referenced files (Xrefs, images, data links, sheet sets)
    • Include material textures
    • Include raster images
  5. File format: AutoCAD LT 2018 Drawing Format (ensures 2019–2026 compatibility).
  6. Bind external references (Bind type: Bind).
  7. Output type: ZIP.
  8. Path options: Place all files in one folder.
  9. Save and generate ZIP.
  10. Send ZIP to recipient.
Receiver Checklist
  1. Save ZIP locally.
  2. Extract ZIP to a folder (Right‑click → Extract All…).
  3. Install any .ttf/.otf (Right‑click → Install for all users).
  4. Copy .ctb to:
    C:\Users<UserName>\AppData\Roaming\Autodesk\AutoCAD LT 2026\R32\enu\Plotters\Plot Styles
  5. Open DWG from the extracted folder.
Where are SHX, TTF, and CTB files found?
  • AutoCAD SHX Fonts: C:\Program Files\Autodesk\AutoCAD LT 20XX\Fonts
  • Windows TTF/OTF Fonts: C:\Windows\Fonts
  • Plot Styles (CTB): C:\Users<UserName>\AppData\Roaming\Autodesk\AutoCAD LT 20XX\enu\Plotters\Plot Styles
Full Length eTransmit Instructions
  1. Step 1 – Open the DWG.
  2. Step 2 – Type ETRANSMIT and press Enter.
  3. Step 3 – Create a new setup: Full_Transmit_With_Fonts (Transmittal Setups → New → Continue).
  4. Step 4 – Modify setup:
    • Transmittal File Type: ZIP
    • Path Options: Place all files in one folder
    • File Format: AutoCAD LT 2018 Drawing Format
    • Bind External References: Bind (Bind type: Bind)
    • Include Options (all checked): fonts, plot style tables, material textures, all referenced Xrefs, photometric web files, data link files, sheet set data files, raster images
    • Optional description: “Transmits everything including fonts. Saved to 2018 format.”
  5. Step 5 – Generate ZIP (verify included files list, then OK).
  6. Step 6 – Send ZIP to recipient.

Layer Manager

Organize geometry into named layers that control color, linetype, lineweight, and plot behavior.

Key Commands & Controls
  • LAYER — Layer Properties Manager
  • Layer controls: On/Off, Freeze/Thaw, Lock/Unlock, Color, Linetype
  • Use CTB plot styles to map colors → lineweights

Common Layers & Styles

  • TRACES — black, 0.25 mm (≈ 5 mil)
  • FIDUCIALS — red, 0.10 mm (thin)
  • ADHESIVE — grey, 0.10 mm (thin)
  • NOTES — light / non‑plot (e.g., green 0.09 mm)

Helpful Tools

open Layer Properties

remap objects to layer styles

Best Practices
  • Create a layer for each type of drawing content (TRACES, FIDUCIALS, ADHESIVE, NOTES).
  • Freeze or turn off irrelevant layers per Layout / viewport.
  • Keep lineweights consistent via CTB; use Layer States for repeatability.

Layer States (Saved Configurations)

Save/restore a snapshot of layer visibility, freeze/thaw, colors, and lineweights. Use one‑click presets like “Silver Traces Layout”.

Create a Layer State
  1. Adjust layers (On/Off, Freeze/Thaw, Color, etc.) as desired.
  2. Run LAYERSTATE → New → name it (e.g., SILVER_TRACES_LAYOUT).
  3. Save.
Examples & Usage
  • “Silver Traces Layout” → TRACES, FIDUCIALS, NOTES
  • “Adhesive Layout” → ADHESIVE, FIDUCIALS, NOTES
  • “Final Assembly” → ALL layers

Restore via Layer States Manager in Model Space or in each viewport. Multiple Layouts can share the same Layer State.

Restore per Viewport
  1. Double‑click inside a viewport to activate it.
  2. Run LAYERSTATE → select preset → Restore.
  3. Freeze/Thaw in that viewport without affecting others.
Export/Import .LAS

Use LAYERSTATE → Export/Import to share presets across drawings.

Layouts (Paper Space)

Layouts represent printable sheets. Model Space = 1:1 design; Layouts = paper with border/title block + scaled viewports.

  1. Switch to a Layout tab.
  2. Insert or Xref your Title Block template.
  3. Create viewport (MV) and set scale (1:2, 1:5, etc.).
  4. Activate viewport, Freeze/Thaw per‑sheet layers, then restore the correct Layer State.

Each Layout becomes a page in the final PDF. Use different presets per Layout.

Block Editor (BEDIT)

Edit reusable blocks (title block, symbols) in isolation and update all instances at once.

  1. Run BEDIT → choose block (e.g., TITLE_BLOCK).
  2. Add/modify geometry, insert attributes, or place fields.
  3. Save & Close → ATTSYNC to refresh placed blocks.

Attributes

Attributes are editable text fields inside a block (e.g., REV, PROJECT, DRAWN_BY, DATE). They can be edited and updated across Layouts.

Create Attributes
  1. Command: ATTDEF
  2. Fill in:
    • Tag (e.g., REV, PROJECT, DRAWN_BY)
    • Prompt (shown when editing)
    • Default (e.g., A)
  3. Place the attribute inside the title block.
Edit & Sync
  • ATTEDIT / EATTEDIT — edit values
  • BATTMAN — batch manage attributes
  • ATTSYNC — sync block definition & instances

Title Block with Attributes

  1. Draw border/boxes/logo in Paper Space.
  2. Create attributes with ATTDEF: REV, PROJECT, DATE, DRAWN_BY.
  3. Select all → BLOCK → name TITLE_BLOCK.
  4. Insert into each Layout and fill fields.

Sheet Set Manager (Advanced)

Manage multiple Layouts as one project and drive common fields like REV from a single place.

Setup
  1. Run SHEETSET → create new Sheet Set.
  2. Add drawing Layouts.
  3. Link title block attributes to fields (e.g., CurrentSheetRevision).
Update REV (Preferred)
  1. Sheet Set Properties → change CurrentSheetRevision from A → C.
  2. REGENALL → publish.
Link REV Attribute to Sheet Set (one‑time)
  1. BEDIT the title block → select REV attribute.
  2. Insert Field → Category: SheetSet → Field: CurrentSheetRevision (or custom property).
  3. Save & Close → ATTSYNC all instances.

Printing Multiple Layouts

  1. Use PUBLISH.
  2. Select all Layouts.
  3. Output to PDF (single multi‑page file).
  4. Each Layout prints as a separate page; verify lineweights and title block data.

Appendix A — Updating Revision Numbers (A → C)

Method 1: Title Block with Attributes (Basic)
  1. ATTEDIT (or EATTEDIT).
  2. Select the title block attribute REV.
  3. Change value from A → C.
  4. Repeat in each Layout if not linked.
  5. ATTSYNC to refresh.
  6. Save and PUBLISH to PDF.
Method 2: Sheet Set Manager (Advanced, Preferred)
  1. SHEETSET → open Sheet Set.
  2. Right‑click Sheet Set name → Properties.
  3. Edit CurrentSheetRevision from A → C.
  4. OK → REGENALL.
  5. All Layouts update automatically.
Setup: Link REV to Sheet Set (one‑time)
  1. BEDIT title block → select REV attribute.
  2. Insert Field → SheetSet → CurrentSheetRevision.
  3. Save & Close → ATTSYNC.
Final Step: Reprinting
  1. Use PUBLISH to select all Layouts.
  2. Output to PDF → send updated package to customer.

Also update DATE and change log as needed; project properties can be updated via Sheet Set Manager.

Appendix B — Layer States Management

Presets for common configurations like Silver Traces, Adhesive, Final Assembly.

  1. Set layers → LAYERSTATE → New → name.
  2. Activate a viewport → restore preset.
  3. Export/Import .LAS to share across drawings.

Appendix C — Worked Example (Existing + New)

Part 1: Organizing an Existing Design

Scenario: Drawing has traces, fiducials, adhesive outlines, and notes mixed on random layers.

Goal: Reorganize geometry, apply Layer States, build Layouts, insert Title Block with attributes, publish multi‑page PDF.

  1. Identify Geometry Categories — TRACES, FIDUCIALS, ADHESIVE, NOTES.
  2. Create Standard Layers — LAYER:
    • TRACES (black, 0.25 mm)
    • FIDUCIALS (red, 0.10 mm)
    • ADHESIVE (grey, 0.10 mm)
    • NOTES (green, 0.09 mm, non‑plot optional)
  3. Move Objects — use MATCHPROP or manual reassignment.
  4. Create Layer States — LAYERSTATE:
    • SILVER_TRACES_LAYOUT → TRACES + FIDUCIALS + NOTES
    • ADHESIVE_LAYOUT → ADHESIVE + FIDUCIALS + NOTES
    • FINAL_ASSEMBLY → ALL layers
  5. Create Layouts — insert Title Block; viewport; scale; restore appropriate Layer State. Rename:
    • Layout “Silver Traces”
    • Layout “Adhesive”
    • Layout “Final Assembly”
  6. Insert Title Block Attributes — REV/PROJECT/DRAWN_BY/DATE → ATTSYNC; set REV = A.
  7. Plot Multi‑Page PDF — PUBLISH → check each page.
Part 2: Creating a New Design
  1. Start from Master Template — ACAD-2019-LITE-MASTER TEMPLATE_10-16-20.
  2. Draw in Model Space — TRACES / FIDUCIALS / ADHESIVE / NOTES layers.
  3. Create Layer States — mirror Part 1 for consistency.
  4. Create Layouts — Title Block + viewport + restore state for each sheet.
  5. Fill Title Block Attributes — e.g., PROJECT = “FHE Smart Label”, REV = A, DATE = today, DRAWN_BY = engineer name.
  6. Optional (Advanced): Link to Sheet Set — SHEETSET; add Layouts; link REV to CurrentSheetRevision.
  7. Publish — PUBLISH to PDF package.
Part 3: Benefits of This Workflow
  • Consistency: same structure across drawings.
  • Scalability: rapid sheet creation using Layer States.
  • Efficiency: REV/project info updated once via Sheet Set.
  • Clarity: layers and Layouts easier for new team members.
  • Professional Output: clean multi‑page PDFs with consistent lineweights.

Appendix D — Missing Files Troubleshooting

Common File Types
  • SHX: AutoCAD‑specific fonts (Program Files\Autodesk\AutoCAD LT\Fonts).
  • TTF/OTF: Windows fonts (C:\Windows\Fonts).
  • CTB: Plot style tables (AppData\Roaming\Autodesk\AutoCAD LT\Plotters\Plot Styles).
If a file is listed as “COULD NOT BE LOCATED”
  1. Note the missing filename (e.g., Futura Md BT.ttf or TT0201M.shx).
  2. Ask a co‑worker to search their system in the locations above and send the file.
  3. Once received:
    • SHX → C:\Program Files\Autodesk\AutoCAD LT 20XX\Fonts
    • TTF/OTF → Right‑click → Install for all users
    • CTB → C:\Users<UserName>\AppData\Roaming\Autodesk\AutoCAD LT 20XX\enu\Plotters\Plot Styles
  4. Restart AutoCAD LT.
Example: eTransmit report shows missing 
The following files could not be located:
   

TT0200M.ttf
TT0201M.ttf
Futura Md BT.ttf
color-8mil.ctb

Solution: Request those fonts/CTB from a co‑worker and install/copy to the proper directories (see above).

Final Note

If fonts are missing and cannot be obtained, AutoCAD will substitute the FONTALT font (usually simplex.shx). This causes text to appear stretched or overflow boxes—always install the correct fonts to avoid layout issues.
